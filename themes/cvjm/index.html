Starter 960 Grid Theme for Concrete5

Download the packaged theme or View it in the Concrete5 Marketplace

- 26kb, .zip, Version 1.1

What is this?

This theme will enable developers to have a strong starting point when making a website. The theme is based off of the 960 Grid Systemframework which enables rapid development of cross-browser complient websites. The theme includes:

  • default.php
  • view.php
  • reset.css (from Eric Meyer via the 960 Framework)
  • text.css (included with the 960 Framework)
  • 960.css
  • main.css (main stylesheet for Concrete5 with a customize_miscellenious declaration)
  • typography.css (with a customize_miscellenious declaration)
  • thumbnail.png
  • description.txt
  • ie6.css, ie7.css, & ie8.css with conditional comments in /elements/header.php

The theme includes the following regions:

  • Automatically displays site title
  • Header Nav (the main automatically generating navigation)
  • "Main" Editable Region
  • "Sidebar" Editable Region
  • Info from the default Concrete5 install such as the site name, login, etc.
  • "Footer" Editable Region

Finally, this theme is licenced under the same licence as Concrete5. I hope to make this theme publicly available to developers. If you have any questions or suggestions for this theme, please send me a messagevia the Concrete5.org website. I welcome any suggestions.

Where can I get help with the 960 Grid System?

You could start by checking out the 960 Grid System Homepage. The homepage has templates, resources, a showcase with impressive examples, and much more.

You can also check out this awesome video tutorialabout how to use the 960 Grid System .

I downloaded this package and want to install it manually. How do I do that?

All you have to do is connect to your server via FTP, then upload the folder (unzipped) to your /packages directory. Afterward, go to "Add Functionality" in the dashboard and you should see the theme in the right column. Click install. Then, go to "Pages & Themes" to activate the theme.
